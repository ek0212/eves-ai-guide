Preference Timeline

Late 2022

ChatGPT start

First default once ChatGPT came out.

Feb 2026

Opus 4.6 phase

Switched to Claude Opus 4.6. The 1M context version was especially strong for large coding and source-heavy work.

Early May 2026

Codex shift

Codex plus GPT-5.5 became the coding default because the harness, worktrees, and repo UX were better.

Now

Opus back for coding

Claude Opus is the coding default again. Codex plus GPT-5.5 high stays in rotation for prototyping and automations. NotebookLM for source-grounded learning and podcasts.
